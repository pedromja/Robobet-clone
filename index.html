✈️ Configurar Telegram

Para receber notificações no Telegram, precisa de criar um bot no Telegram e obter o token. Siga os passos abaixo:

  1. Abra o Telegram e procure por @BotFather
  2. Envie /newbot e siga as instruções
  3. Copie o token que receber
  4. Comece uma conversa com o seu bot
  5. Clique em "Obter Chat ID" abaixo
🔑 Configurar API

Escolha a API de dados para futebol. Recomendamos API-Football (gratuita até 100 req/dia). Registe-se em api-football.com ou rapidapi.com para obter a sua chave.
